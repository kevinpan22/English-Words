# 📖 翻译优化说明

## 🎯 优化目标

为小学、中学生提供**简短、常用、易懂**的中文翻译。

---

## ✨ 优化策略

### 1️⃣ 常用单词预设翻译（最快）

**200+ 个常用单词**已预设简短翻译，**无需联网**，瞬间返回结果。

**覆盖范围**：
- 🐾 动物（cat→猫, dog→狗, bird→鸟...）
- 🍎 水果（apple→苹果, banana→香蕉...）
- 🎨 颜色（red→红色, blue→蓝色...）
- 🔢 数字（one→一, two→二...）
- 👨‍👩‍👧 家庭（father→父亲, mother→母亲...）
- 🏃 常用动词（go→去, eat→吃, run→跑...）
- 📚 学校（school→学校, teacher→老师...）
- ⏰ 时间（today→今天, tomorrow→明天...）

**示例**：
```javascript
apple → 苹果（预设，无需API）
happy → 开心的（预设，无需API）
school → 学校（预设，无需API）
```

---

### 2️⃣ 智能简化（API调用）

对于**不在预设词典**的单词，使用智能简化算法：

#### 步骤1：简化英文定义
```
原始定义：
"a large plant-eating domesticated mammal with solid hoofs and a flowing mane and tail, used for riding, racing, and to carry and pull loads"

简化后：
"large plant-eating mammal"
```

#### 步骤2：翻译简化后的内容
```
翻译：大型食草哺乳动物
```

#### 步骤3：进一步简化中文
```
最终：大型食草动物（限制15字以内）
```

---

### 3️⃣ 简化规则

**英文简化**：
- ✂️ 移除括号内容：`(especially in the past)` → 删除
- ✂️ 移除从句：`, especially...` → 删除
- ✂️ 只保留第一句：`sentence1. sentence2` → 只要 sentence1
- ✂️ 移除冠词：`a dog` → `dog`
- ✂️ 限制长度：最多50字符

**中文简化**：
- ✂️ 移除括号
- ✂️ 只保留第一部分（逗号前）
- ✂️ 限制长度：最多15个汉字

## 🔧 如何添加更多常用单词

### 方法1：直接编辑代码

打开 `app.js`，找到 `commonWordsMeanings` 对象：

```javascript
const commonWordsMeanings = {
    'apple': '苹果',
    'banana': '香蕉',
    // 在这里添加新单词
    'car': '汽车',      // 添加这行
    'bus': '公共汽车',  // 添加这行
    // ...
};
```

### 方法2：批量添加（推荐）

创建一个 `custom-words.js` 文件：

```javascript
// 添加到 commonWordsMeanings
Object.assign(commonWordsMeanings, {
    // 你的自定义单词
    'car': '汽车',
    'bus': '公共汽车',
    'train': '火车',
    'plane': '飞机',
    // ...
});
```

然后在 `index.html` 中引入：
```html
<script src="custom-words.js"></script>
```

## 🙋 常见问题

### Q: 预设词典有多少个单词？
A: 当前约 **200个** 常用单词，覆盖小学核心词汇。

### Q: 可以完全离线使用吗？
A: 预设词典中的单词可以离线使用，其他单词需要联网查询音标和翻译。

### Q: 翻译不准确怎么办？

### Q: 如何批量添加四六级词汇？
