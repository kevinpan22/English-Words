## 💡 如何进一步优化

### 方法1：添加学科词汇
打开 `app.js`，在 `commonWordsMeanings` 中添加：
```javascript
// 数学
'math': '数学',
'plus': '加',
'minus': '减',
'multiply': '乘',
'divide': '除',

// 科学
'science': '科学',
'experiment': '实验',
'theory': '理论',
```

### 方法2：批量导入四六级词汇
创建 `cet-words.js`：
```javascript
Object.assign(commonWordsMeanings, {
    'abandon': '放弃',
    'abstract': '抽象的',
    'academic': '学术的',
    // ... 更多词汇
});
```

### 方法3：使用用户反馈优化
遇到翻译不准的单词，记录并添加到词典。
